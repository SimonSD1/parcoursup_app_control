Tutoriels

Tuto pour les ordis :

  1. Sélectionner les actions voulues puis appuyer sur envoyer
  2. Ne rien sélectionner puis appuyer sur envoyer pour pouvoir refaire les actions

Tuto pour changer le code :

  1. Vérifier que l'état est sur off 
  2. Ecrire le code dans la zone de texte
  3. Cocher activation et appuyer sur envoyer
  4. Ne rien sélectioner puis appuyer sur envoyer
